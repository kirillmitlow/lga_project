# lga_project

Моделирование двумерного потока жидкости с использованием клеточного газового автомата (Lattice Gas Automata, LGA) на языке C.

## Описание

Данный проект реализует вычислительную модель LGA для симуляции гидродинамики в дискретной решётке. В этой модели частицы перемещаются по фиксированным направлениям, взаимодействуют друг с другом и сталкиваются с препятствиями, что позволяет воспроизводить физические процессы макроскопического течения.

## Основной функционал

* **Модель решётки**: Используется гексагональная (triangular) решётка. Для удобства вычислений она хранится в прямоугольном 2D-массиве с offset-layout, где чётные и нечётные строки сдвинуты относительно друг друга на 1/2 шага.
* **Состояния частиц**: Каждая клетка (`Cell`) использует битовую маску для отслеживания 7 состояний (6 направлений движения и 1 покоящаяся rest-частица), а также содержит флаг принадлежности к препятствию.
* **Правила эволюции**:
    * **Collision (Столкновения)**: Реализованы правила FHP-II с сохранением массы и импульса. Обрабатываются двухчастичные лобовые столкновения (с равновероятным выбором из двух исходов для изотропии), трёхчастичные столкновения и взаимодействия с rest-частицей. Для оптимизации используется lookup-таблица на 128 возможных состояний.
    * **Streaming (Перемещение)**: Перемещение частиц по 6 направлениям с учётом смещения (offset) рядов.
* **Граничные условия**:
    * Верхняя и нижняя стенки канала реализуют простое отражение (`bounce-back`).
    * Левая граница работает как источник (inlet), генерируя частицы с заданной плотностью и скоростью, направленной вправо.
    * Правая граница работает как свободный выход (outlet).
* **Препятствие**: В центре канала расположен круглый цилиндр заданного диаметра.
* **Вывод и визуализация**:
    * Сохранение кадров симуляции в виде цветных PPM-файлов через заданные интервалы.
    * Расчет и вывод средней плотности системы.
    * Сохранение итогового состояния поля в бинарный файл `grid.bin`.

## Сборка и запуск

Для сборки проекта используйте компилятор C или настроенный `Makefile`. 

```bash
make
# или
gcc -o lga main.c
```

Запуск программы осуществляется через командную строку с передачей конфигурационных параметров:

```bash
./lga -width 256 -height 160 -iters 1000 -rho_in 1.5 -diameter 20 -seed 42 -viz_interval 100
```

### Параметры командной строки
* `width` / `height` — размеры сетки по ширине и высоте.
* `iters` — количество итераций симуляции.
* `rho_in` — начальная плотность частиц на левой границе.
* `diameter` — диаметр препятствия в центре канала.
* `seed` — сид генерации случайных чисел для инициализации решетки.
* `viz_interval` — частота сохранения визуализации в PPM.
